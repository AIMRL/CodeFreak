CodeFreak is a programming website, which will target programming competitions, programming projects, issue raising and social interaction. For programming competition, it will provide an interface to manage a programming competition event. For programming projects, it will provide general interface to communicate between teachers and students. It will also provide facility to raise any issue, where you can expect reply of both university fellows and teachers. Furthermore, this solution also offers a social network for teachers and students. Thus, this project is aimed to provide a general platform to address various problems.
